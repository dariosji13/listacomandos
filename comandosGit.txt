git init: para crear un nuevo repositorio
git add: con este se pasa el proyecto  del working directory al stagin area
git statis: para ver en que estado se encuentra nuestro proyecto
git commit: para pasar los archivos al repositorio
git push: para subirlo a un repositorio remoto
git pull: para acceder a los cambios hechos en el proyecto
git clone: para hacer una copia exacta del proyecto desde el ordenador centar  o la nube al tuyo
git reset: es para eliminar o deshacer un archivo
git checkout -- nombre archivo: es para deshacer los cambios hechos a  un archivo
git log: permite ver el historial de los diferente commit que se han hechos
git branch: para ver las ramas que hay y en cual se encuentra 
git branch nombre de la rama: para craer una rama
git checkout -b nombre de la rama: crea una rama y se ubica en ella
git checkout -d nombre de la rama: para elimar la rama
git branch -- all: para ver todas las ramas incluidas las remotas
